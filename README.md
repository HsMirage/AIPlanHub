## 平台推荐与提醒

**推荐**

- **智谱AI** — 独家 GLM-5.1 模型，免费 MCP 调用额度，20+ 编程工具支持，季付/年付有折扣
- **z.ai** — 智谱国际版，GLM-5.1/GLM-5 全模型支持，美元计费，季付 -10% / 年付 -30%，MCP 免费调用
- **MiniMax** — 定价最低无需抢购，独家 MiniMax-M2.7，已升级 Token Plan，Plus 及以上赠多模态额度
- **天翼云** — 除智谱官方外唯一支持 GLM-5.1，GLM 全家桶覆盖最完整，GLM-5-Turbo 实时测速表现靠前
- **优云** — 允许 API 调用，支持 Claude Code / OpenClaw / Codex CLI / CherryStudio，另有 ¥6.9 起一次性按量包，第三方测速整体靠前
- **无问芯穹** — 多模型聚合平台（DeepSeek/Kimi/GLM/MiniMax），Lite 首月 ¥19.9

**注意**

- **字节·方舟** — ⚠ 双层计费：名义按调用次数，实际 Token 消耗大会被按 2-3 次甚至更多次扣费，计费不透明，无用量明细（蓝点网/V2EX/微博多源证实）
- **天翼云** — 高阶 GLM 模型在高峰期按 3 倍、非高峰期按 2 倍抵扣额度，实际可用量会明显低于表面档位；当前页面长期显示售罄/补货
- **京东云** — 仅限 AI 编程工具交互式使用，禁止 API 调用；社区有“容易截断、有点卡”和“未到 5h 上限仍触发 rate_limit”的反馈
- **联通云** — 当前资源紧张，触发限流时会自动切到负载更轻模型；Kimi 通道第三方测速波动很大，且禁止 API 调用
- **移动云** — 仅支持 MiniMax-M2.5，且目前只开放华北-呼和浩特与湖北-武汉资源池订购；Coding Plan 同样禁止 API 调用
- **小米·MiMo** — Credit ≠ Token（Pro 2x / Omni 1x），¥39 仅 30M token（Pro），性价比远低于同行，兼容 Claude Code 但定价无竞争力
- **腾讯·Token** — 仅限 AI 工具使用，禁止 API 调用（违者封禁）；暂不支持多模态

## 说明

- 季付均价 = 季付总价 ÷ 3；年付均价 = 年付总价 ÷ 12
- 5h 请求数基于「1 prompt ≈ 15 次调用」估算
- Token量列：展示月额度或日额度；单位 M = 百万 Tokens
- 优云为积分制，实际可用 Token 受模型倍率影响；页面另提供一次性按量包
- Midjourney / Suno / Udio / Ace Studio / z.ai 价格按 $1 ≈ ¥7.0 估算，实际以美元结算
- 小米·MiMo Credit 倍率：MiMo-V2-Omni 256k→1x, MiMo-V2-Pro 256k→2x, MiMo-V2-Pro 256k~1M→4x, MiMo-V2-TTS→0x（限时免费）
- 腾讯·Token Plan 仅限 AI 工具（Claude Code/Cursor/OpenClaw 等）使用，禁止 API 调用
- 部分平台通过邀请链接开通可享额外折扣
- 数据仅供参考，以各平台官方公告为准
